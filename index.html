Power BI

Survey Responses Analysis

This project evaluated satisfaction levels across key IT services. Python was used to clean and standardize survey responses, convert qualitative ratings into numeric values, and merge the data with a master contact list to identify business group (BG) affiliations. It then calculated average satisfaction scores and response counts by BG. The resulting insights offered a clear, data-driven view of service quality, enabling targeted improvements in IT service delivery.
View Project

Personal Finance Tracker

This project is a command-line based personal finance tracker that allows users to log income and expenses, analyze spending patterns, and visualize financial trends over time. Built in Python, it uses CSV for persistent data storage and pandas for data manipulation. Users can input transactions interactively, with built-in validation for dates, categories, and amounts. The tool summarizes financial activity over a specified date range and provides optional visualizations using matplotlib. This project promotes financial awareness through simple yet insightful tracking and analysis.
